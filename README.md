# AmigaJoystickTesterPCB

Simple PCB for testing a 9pin Amiga Joystick.

Components:
* Resistors 220 OHM
* Leds 2V - 20mA
* D-SUB 9 pin PCB Male Connector
* Micro USB Type B Female 5 Pin Connector plug adapter
